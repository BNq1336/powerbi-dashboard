# Dashboard Visualization – Northwind

## Opis projektu

Interaktywny dashboard biznesowy stworzony w ‎Microsoft Power BI na podstawie bazy Northwind. Projekt skupia się na analizie sprzedaży, marży, zamówień, klientów oraz prognozowaniu wyników.

## Funkcjonalności

* analiza wyników sprzedaży,
* segmentacja klientów i krajów,
* analiza marży,
* monitorowanie statusów zamówień,
* analiza przewoźników,
* wizualizacje prognoz sprzedaży,
* mapa klientów.

## Strony dashboardu

* Overview sprzedaży,
* Analiza krajów,
* Analiza przewoźników,
* Analiza zamówień,
* Analiza miast klientów,
* Statusy zamówień,
* Analiza marży,
* Forecast sprzedaży,
* Mapa klientów.

## Technologie

* Microsoft Power BI
* Power Query
* DAX
* Data Modeling

## Zawartość repozytorium

* plik `.pbix`,
* raporty,
* przykładowy workflow Business Intelligence.

## Uwagi

Projekt został przygotowany w celach edukacyjnych oraz portfolio.
